
# Money Leash

Don't let your money run wild. Keep it on a leash!


# PLEASE NOTE

This project, developed in ionic 1 and angularjs, is no longer being supported. It has been replaced for MoneyLeash2, a complete rewrite from the ground up using ionic 2 and angular 2.

For more information visit [MoneyLeash2](https://github.com/gigocabrera/MoneyLeash2)
